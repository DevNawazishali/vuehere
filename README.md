# BombCryptoWeb

## Project setup

```
npm install

*需要国际化，安装 vue-i18n 以及 yaml-loader
npm install vue-i18n -D
npm install yaml-loader -S

*web3
npm install web3 -D

*cookies-js
npm install cookies-js -D

```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
